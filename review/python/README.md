## [PEP498: f-Strings](https://www.python.org/dev/peps/pep-0498/)
```python
f'{2:b}'  # '10'
f'{2:#b}'  # '0b10'
date = datetime.date(1991, 10, 12)
f'{date} was on a {date:%A}'  # '1991-10-12 was on a Saturday'
```
