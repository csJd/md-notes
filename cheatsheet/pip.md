# MY PIP CHEAT SHEET

### Basic
```sh
pip -V,--version
pip list  # list packages
pip search keyword  # Search PyPI for keyword
pip show pkg  # Display package info
```

### Install

``` sh
pip install pkg
pip install pkg==1.0,pkg>=1.0
pip install [-U,--upgrade] pkg
pip install --force-reinstall pkg
pip uninstall pkg
```
